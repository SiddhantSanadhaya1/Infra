## Running Tests

### Set Base URL

Configure the API endpoint using the `BASE_URL` environment variable:

```bash

```

For deployed environments:
```bash

```

## Troubleshooting

### Tests Failing with Connection Error
- Ensure the API service is running: `curl $BASE_URL/health`
- Check that BASE_URL is set correctly
- Verify database connectivity

### Tests Timing Out
- Check API service logs for errors
- Verify database performance
- Ensure AWS services (S3, SQS) are accessible

### Random Test Failures
- Tests use unique emails to avoid conflicts
- Each test is independent (no shared state)
- Check for database transaction issues
