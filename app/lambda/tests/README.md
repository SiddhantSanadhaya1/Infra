# Lambda Worker Unit Tests

Comprehensive unit tests for the InsureCo Insurance Lambda background worker.

## Test Framework

- **Framework**: pytest
- **Coverage**: Job handlers, fraud scoring, Lambda entry point
- **Mocking**: unittest.mock for AWS services (DynamoDB, SNS)

## Running Tests

```bash
# Install test dependencies
pip install -r requirements-test.txt

# Run all tests
pytest

# Run with coverage
pytest --cov=worker --cov-report=html

# Run specific test class
pytest tests/test_worker.py::TestCalculateFraudScore
```
